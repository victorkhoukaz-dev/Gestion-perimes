Guide d'Utilisation Interactif

Ce guide vous explique en détail comment utiliser l'application au quotidien pour vos tournées de saisie, vos retraits mensuels et la gestion de vos codes-barres.

💡 Un processus simple en 2 étapes !
Une tournée complète semestrielle (pour répertorier les produits qui expirent d'ici 6 mois), et le retrait mensuel (pour retirer physiquement les produits qui viennent à expiration).

Cet onglet sert à enregistrer les produits lors de votre tournée de repérage des expirations :

  • Sélectionnez la section : Le menu conserve automatiquement votre choix d'un produit à l'autre pour vous éviter des clics.
  • Scannez ou Saisissez : Placez le focus dans le champ "DIN ou code UPC". Vous pouvez directement scanner la boîte avec votre scanner ou tapez le DIN.
  • Auto-remplissage : Si le produit est connu, le Nom, Dosage et Format se pré-remplissent, et le curseur se place directement sur le champ Quantité. Saisissez la quantité et appuyez sur Entrée pour enregistrer !
  • Pastilles de couleur : L'application vous affiche un aperçu visuel du sticker à apposer sur la boîte selon la date d'expiration entrée.
💡 Conseil de pro : Entrez vos initiales dans le champ "Technicien". L'application les mémorisera pour toute votre session de travail.

Cet onglet est votre feuille de route au début de chaque mois pour retirer physiquement les boîtes des tablettes :

  • Il liste uniquement les produits actifs qui expirent durant le mois en cours.
  • Les produits sont regroupés et triés par section du laboratoire pour optimiser vos déplacements.
  • Actions rapides : Cliquez sur ✅ Retirer si vous avez trouvé le produit et l'avez jeté/renvoyé, ou sur 📦 Vendu s'il n'est plus sur les tablettes.
  • Impression et signatures : Utilisez le bouton "Imprimer la fiche" pour générer une fiche papier propre à faire signer par le pharmacien vérificateur.

Si vous scannez une boîte dont le code-barres n'est pas encore connu de l'application, vous pouvez l'associer en direct :

  1. Scannez le code-barres. Un message s'affiche indiquant que le code est inconnu.
  2. Saisissez le DIN du produit dans la case (qui s'est vidée automatiquement).
  3. Sélectionnez le produit dans la liste d'auto-complétion. Un bandeau vert confirme que l'association est planifiée.
  4. Saisissez la quantité et cliquez sur Enregistrer. L'association s'enregistre de manière permanente. La prochaine fois que vous scannerez ce produit, il sera reconnu instantanément !
⚠️ Sécurité : Si vous scannez un code inconnu par erreur, cliquez sur le bouton rouge "Annuler l'association" sous la case de saisie pour annuler l'opération.

Dans l'onglet Configuration, vous pouvez mettre à jour les données :

  • Importation du fichier d'inventaire : Importez le catalogue complet des produits de votre pharmacie (format CSV). Cela active l'auto-complétion instantanée de tous les DIN du laboratoire.
  • Association des codes UPC aux DIN : Importez un fichier CSV contenant à la fois UPC et DIN pour populer en masse votre catalogue existant (Utile uniquement si votre fichier d'inventaire initial ne contenait pas de codes UPC).

Bienvenue dans l'application !

Voici un résumé rapide du fonctionnement de la gestion de vos expirés de laboratoire :

1

La Tournée de Saisie (aux 6 mois)

Faire une tournée complète du laboratoire afin d'identifier tous les produits expirant durant les 6 prochains mois. Scannez le code-barres (UPC) ou tapez le DIN, saisissez la quantité, puis apposez la pastille de couleur mensuelle correspondante sur la boîte.

2

Le Retrait Mensuel (Début de chaque mois)

Dans l'onglet Retrait Mensuel, l'application vous montre uniquement les produits sur tablette qui expirent ce mois-ci, triés par section. Retirez-les physiquement des tablettes et cochez Retiré ou Vendu sur l'écran.
